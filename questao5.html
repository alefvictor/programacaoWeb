Java

Java é uma linguagem de programação orientada a objeto desenvolvida na década de 90
por uma equipe de programadores chefiada por James Gosling, na empresa Sun
Microsystems1.

Diferentemente das linguagens convencionais, que são compiladas para código nativo, a
linguagem Java é compilada para um bytecode que é executado por uma máquina virtual.
Atualmente, está versão 7 versão 8.


Exemplo de código:
        public class Olá Mundo{
                    public static void main(String[]args){
                                System.out.println("Olá, Mundo!"); //Imprime na tela a frase
                    }
        }
