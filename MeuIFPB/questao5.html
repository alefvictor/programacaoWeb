Java

Java é uma linguagem de programação orientada a objeto desenvolvida na década de 90 por uma equipe de programadores chefiada por James Gosling, na empresa Sun Microsystems¹.

Diferentemente das linguagens convencionais que são compiladas para código nativo, a linguagem Java é compilada para um bytecode que é executado por uma máquina virtual². Atualmente, está na versão7 versão 8.
